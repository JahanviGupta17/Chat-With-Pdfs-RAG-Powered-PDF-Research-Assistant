# Chat-With-Pdfs-RAG-Powered-PDF-Research-Assistant
A Streamlit-based research assistant that uses Hugging Face LLMs, Pinecone vector database, and FAISS-like embeddings to:  Ingest multiple PDFs and large research papers.  Split text into chunks and create a retrieval-augmented generation (RAG) pipeline.  Summarize content and answer research questions with source citations.
